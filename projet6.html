Passatemps – Web Application

Web • Frontend & Backend • User Experience • Marketplace

Project Overview

Passatemps is a web platform for buying and selling second-hand board games. The goal is to reduce waste and give games a second life, allowing users to sell unused games and make them available to others.

I recently joined this project, which is currently under development, to participate in its management and implementation this year. The focus is on enhancing the user experience, adding new features, and continuing the platform’s development as part of a structured project workflow.

  • Technologies: HTML, CSS, JavaScript, PHP/MySQL (or backend in use)
  • Theme: Second-hand board game marketplace
  • Target users: Young adults and families
  • Features: Intuitive interface, messaging system, ratings and comments

What I Learned

Joining this project allowed me to get involved in a real web application development process, focusing on both the frontend and backend aspects. I am learning how to manage a project workflow, implement new features, and design a user-friendly interface.

This experience also helps me understand the challenges of building a marketplace platform, including user experience, messaging, and moderation of content.
