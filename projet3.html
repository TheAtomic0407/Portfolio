Bulbizness

HTML • CSS • Web App

Project Overview

Bulbizness is a web application designed to help users manage their business tasks efficiently. It features a clean interface with interactive elements created purely with HTML and CSS.

The main goal was to create a responsive and user-friendly design without relying on JavaScript.

  • Language: HTML, CSS
  • Framework/Tools: Bootstrap
  • Concepts: Responsive design, Flexbox/Grid, CSS effects

What I Learned

Through this project, I improved my skills in HTML and CSS, learning how to build a clean, responsive, and user-friendly interface. I also practiced responsive design and using Bootstrap components effectively.

Overall, Bulbizness gave me practical experience in designing a web app interface without JavaScript while maintaining interactivity through HTML/CSS techniques.
